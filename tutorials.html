GTA San Andreas Modding - CLEO Scripting - How to make a cheat code!

In this tutorial i will be demonstrating how with the cleo library and SannyBuilder you can develop your very own cheat codes. First you'll need SannyBuilder, and ofcourse the CLEO library installed on your copy of San Andreas.

STEP-BY-STEP











  • 1. Open SannyBuilder

  • 2. Go to File > New

  • 3. You should now have a blank page open on the program, good!

  • 4. We need to begin our script by declaring what format we want it to be saved in. At the very top of the page type in

    {$CLEO .cs}

    "Okay so? What does this do?"

    This tells the program what format to compile your code into when you hit the compile button. In this case it's .cs which stands for CLEO Script.

  • 5. You need to create a thread, name it whatever you like. For instance i decided to name mine 'teles' as i am going to make a teleportation cheat code.

    e.g: 03A4: name_thread 'teles'

  • 6. Now we must create a label, to create a label just move down from your thread you created and type :'insertyourthreadsnamehere' (without quotations) Within labels we put in opcodes, functions and all the code which will make up what will become our script. My label is :tele

  • 7. Let's get started on the cheat itself, in my case it's going to be a teleport.
    Type:

    :teles
    wait 0
    if
    0ADC: test_cheat "VENTURASHEREICOME"
    then
    else_jump @teles


    What this is basically saying is if the player doesn't test the cheat string ("VENTURASHEREICOME") then just stay in the loop waiting for the moment the player types it.

  • 8. Let's expand our code now we have a condition in place.

    Add:

    Actor.PutAt($PLAYER_ACTOR, X coordinate, Y coord, Z coord)
    03E5: show_text_box "CHEAT ACTIVATED"
    end

    Now our overall code is basically saying wait for the player to test the string, if the string IS tested then place the player ($PLAYER_ACTOR, this is the players global handle) at the specified coordinates and show a text box. To figure out what coordinates your teleport is going to be you must run GTA SA, head to your location. ALT+TAB out of the game and open SannyBuilder, hit CTRL+ALT+1 and it will open a dialog box showing your coordinates, hit copy and paste the coordinates in the Actor.PutAt($PLAYER_ACTOR, _______,______,______)

  • 9. Now compile your code by hitting the compile button, which is two buttons from the left of the options cog. Make sure you compile the file into your CLEO folder!

  • 10. Test the cheat and enjoy! You can edit the outcomes of inputting the string to make your own cheats if you followed this guide.
